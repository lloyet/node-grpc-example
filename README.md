# GRPC TYPESCRIPT

gRPC example for Node.js in Typescript

# Installation

```sh
# Install packages
npm install

# Generate .proto files
npm run proto:gen

# Run in dev
npm run dev

# Build app
npm run build
```

# Env

> Create a .env to store environmment variables

| ENV VAR    | Description         |
| ---------- | ------------------- |
| HOST       | Hostname            |
| PORT       | Port number         |
| PROTO_PATH | File Path to .proto |
